CSS: What Is It?

CSS = Cascading Style Sheets

CSS is a "style sheet language" that lets you style the elements on your page.

CSS works in conjunction with HTML, but is not HTML itself.

Cascading:

  • Inheritance
  • Conflict resolution
    • Last Rule
    • Specificity
    • Important

CSS: What Can It Do?

Change color, spacing, position, and size

Attributes That Aid CSS

Optional attributes common to all HTML elements

  • id assigns a unique identifier to the element 
    
                <p id="footer">Copyright 2011</p>
  • class classifies an element, multiple elements can share a class. An element can have multiple classes. 
    
                <p class="warning red">Run Away!</p>

Adding CSS to an HTML Page


<head>
  <style type="text/css">
    p {
      color: blue;
      font-size: 12px;
    }
  </style>
</head>

Inside <head> element.

Uses <style> tag.

Can only be used in one html file

Property: Color

The color property changes the color of the text.


p {
  color: red;
  color: #ff0000;
  color: rgb(255, 0, 0);
}

Color name

Hexadecimal value

RGB (Red Green Blue) value

The 17 standard colors are: aqua, black, blue, fuchsia, gray, grey, green, lime, maroon, navy, olive, purple, red, silver, teal, white, and yellow.

Property: Background-Color

The background-color property changes the color of the background.


p {
  background-color: black;
  background-color: #000000;
  background-color: rgb(0,0,0);
}

Property: Font-Family

The font-family property deﬁnes which font is used.


p {
  font-family: "Times New Roman";
  font-family: serif;
  font-family: "Arial", sans-serif;
}

Specific font name

Generic name

Property: Font-Size

The font-size property speciﬁes the size of the font.


p {
  font-size: 12px;
  font-size: 1.5em;
  font-size: 100%;
}

Pixels - a dot on the computer screen

"em" - 1em is the current font size

Percentage - percentage of the overall real estate

Property: Fonts (shorthand)


p {
  font-style: italic;
  font-weight: bold;
  font-size: 10px;
  font-family: sans-serif;
}
OR 

p {
  font: italic bold 10px sans-serif;

Connecting CSS to HTML

3 ways

"Inline"

"Embedded"

"External"

Connecting CSS to HTML: Inline


  <p style="color:red">Some text.</p>

Uses the HTML attribute style.

Difficult to use in large projects

Not preferred.

Connecting CSS to HTML: Embedded


<head>
  <style type="text/css">
    p {
      color: blue;
      font-size: 12px;
    }
  </style>
</head>

Inside <head> element.

Uses <style> tag.

Can only be used in one html file

Connecting CSS to HTML: Linked


<head>
  <link rel="stylesheet" type="text/css" href="style.css">
</head>

Shared resource for several pages.

Reduced ﬁle size & bandwidth

Easy to maintain in larger projects.

Preferred by nerds everywhere!

Let's Develop It!

  • Start a new file in your text editor and save it to your html-site folder as style.css
  • Add a link to the file in the head of the index.html page made last time.
    • 
<head>
  <link rel="stylesheet" type="text/css" href="style.css">
</head>
  • Move the CSS you already created to the external file
  • Check your work. Did the styling stay the same?

Selector Priority/Specificity

The more specific a selector is the higher priority it will be given.

  1. !important styles
  2. inline styles
  3. selecting with id
  4. selecting with a class
  5. selecting with a element name

p{
  color:blue;
}
#special{
  color:yellow;
}
.red{
  color:red;
}
.alwaysgreen{
  color:green !important;
}



  <p>I am blue</p>
  <p class="red" style="color:orange;">I am orange!</p>
  <p class="alwaysgreen red" id ="special">I am ALWAYS Green</p>
  <p class="red" id="special">Yellow Paragraph</p>
