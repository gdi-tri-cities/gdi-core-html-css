Inline & Blocking Elements

"Block" elements begin on a new line:

  • paragraphs - <p>
  • headers - <h1>
  • lists - <ul> <li>

"Inline" elements maintain the line flow.

  • anchor/link - <a href>
  • image - <img />
  • emphasis - <em>
  • strong - <strong>

Pseudo-classes: More Style for Links

Changing the format of a link when you hover over it is accomplished by using pseudo-classes.


CSS pseudo-classes are used to select tags in a certain state.

Syntax:


        selector:pseudo-class
        {
          property:value;	
        }
Example: 

        a:link
        {
          text-decoration: none;
        }

Pseudo-classes:
Responding to the User


          a:link    {color:#FF0000;} /* unvisited link */
          a:visited {color:#00FF00;} /* visited link */
          a:hover   {color:#FF00FF;} /* mouse over link */
          a:focus   {color:#FFFF00;} /* holding pointer over link */
          a:active  {color:#0000FF;} /* selected link */

Order of pseudo-classes in stylesheets:

  1. :link
  2. :visited
  3. :hover
  4. :focus
  5. :active

Box Model

Padding

Padding is the space between the border and the content, increasing the total size of the box

Padding - Individual Rules

15 pixels on all sides:


          padding: all;
          padding: 15px;

10 pixels each side, individually assigned:

padding-top: 10px;
          padding-left: 10px;
          padding-right: 10px;
          padding-bottom: 10px;

10 on top, 5 on right, 3 on bottom, 5 on left:

padding: top right bottom left;
          padding: 10px 5px 3px 5px;

10 on top & bottom, 5 on the sides

padding: top/bottom right/left;
        padding: 10px 5px;

Padding - Combined Rule


          padding: 10px 20px 30px 40px;

Border

The edge around the box, specified as "thickness, style, color."

Border Styling

Make all the borders solid red:


          border: 1px solid #ff0000;

Assign each border style individually:


          border-top: 1px solid #000000;
          border-bottom: 2px dotted #000000;
          border-left: 3px dashed #000000;
          border-right: 4px ridge #000000;

Assign each attribute individually:


          border-width: 10px 
          border-style: dotted;
          border-color: #000000;

Border Styles

Border definition

Margin

The transparent area around the box that separates it from other elements.

Margin

15 pixels on all sides:

  margin: 15px;

Each side assigned individually:

  margin-top: 10px;
          margin-bottom: 10px;
          margin-left: 10px;
          margin-right: 10px;

Assign each side in one line

  
          margin: top right bottom left
          margin: 10px 5px 3px 5px;

Assign top & bottom together, and sides together

  
          margin: top/bottom right/left
          margin: 10px 5px;

Auto Margin

If a margin is set to auto on an element that has width, it will take up as much space as possible.

CENTERED


          margin: auto;
          width: 300px;

FLUSH-RIGHT


          margin-left: auto;
          margin-right: 5px;
          width: 300px;
