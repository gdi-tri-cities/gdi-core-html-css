Inline vs Block

So far, we have mostly seen "block" elements

They appear on the next line, like paragraphs



There are also "inline" elements

They appear on the same line that they are written on.

Block & Inline Elements,

  • CSS divides HTML into two types: inline and block.
  • After block elements, browsers render a new line.
  • Inline elements: img, a, em, strong
  • Block elements: p, h1, ul, li, almost everything else

Pseudo-classes, more CSS for links

Changing the format of a link when you hover over it is accomplished by using pseudo-classes.


CSS pseudo-classes are used to add special effects to some selectors.

Syntax: 

selector:pseudo-class
{
  property:value;	
}
Example: 

a:link
{
  text-decoration: none;
}

Pseudo-classes, more CSS for links


  a:link	{color:#FF0000;} /* unvisited link */
  a:visited	{color:#00FF00;} /* visited link */
  a:hover	{color:#FF00FF;} /* mouse over link */
  a:active	{color:#0000FF;} /* selected link */

Note: a:hover MUST come after a:link and a:visited in the CSS definition in order to be effective!

Note: a:active MUST come after a:hover in the CSS definition in order to be effective!

Box Model

Padding

Space between the border and the content

Adds to the total width of the box.

Padding

15 pixels on all sides: 
padding: all;
padding: 15px;
10 pixels each side, assigned individually: 
padding-top: 10px;
padding-left: 10px;
padding-right: 10px;
padding-bottom: 10px;
10 on top, 5 on right, 3 on bottom, 5 on left: 
padding: top right bottom left;
padding: 10px 5px 3px 5px;

10 on top & bottom, 5 on the sides

padding: top/bottom right/left;
padding: 10px 5px;

Padding


  padding: 10px 20px 30px 40px;

Border

The edge around the box, specified as "thickness, style, color."

Border

Make all the borders solid red:


  border: 1px solid #ff0000;

Assign each border individually:


  border-top: 1px solid #000000;
  border-bottom: 2px dotted #000000;
  border-left: 3px dashed #000000;
  border-right: 4px ridge #000000;

Assign each attribute individually:


  border-width: 10px 
  border-style: dotted;
  border-color: #000000;

Border Styles

More on borders

Margin

The transparent area around the box that separates it from other elements.

Margin

15 pixels on all sides:

  margin: 15px;

Each side assigned individually:

  margin-top: 10px;
  margin-bottom: 10px;
  margin-left: 10px;
  margin-right: 10px;

Assign each side in one line

  margin: top right bottom left
  margin: 10px 5px 3px 5px;

Assign top & bottom together, and sides together

  margin: top/bottom right/left
  margin: 10px 5px;

Auto Margin

If a margin is set to auto on a box that has width, it will take up as much space as possible.

CENTERED


  margin: auto;
  width: 300px;

FLUSH-RIGHT


  margin-left: auto;
  margin-right: 5px;
  width: 300px;
